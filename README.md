# Simple 32-bit RISC-V Processor using Verilog

## Overview

This project implements a simplified 32-bit RISC-V style processor in Verilog HDL.

## Features

* 32-bit architecture
* Register File
* Instruction Decode
* Execute Stage
* Arithmetic Operations

  * ADD
  * SUB
  * AND
  * OR

## Tools Used

* Verilog HDL
* EDA Playground
* Icarus Verilog

## Files

* design.sv : CPU Design
* testbench.sv : Verification Testbench

## Results

All directed test cases passed successfully.
